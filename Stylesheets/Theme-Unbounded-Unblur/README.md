## 安装
在网易云内点击设置，滑到最下面，点击打开配置文件夹，将`Unbounded-Unblur.css`放入stylesheets文件夹
#### Stylesheet JSON
```json
,{
    "name":"Unbounded-Unblur",
    "file":"Unbounded-Unblur.css",
    "enabled":false
}
```

## 自定义
自定义背景图：更改对应css文件的第二行为
```css
    background: url("图片地址");
```
